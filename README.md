# Yatube API

API для социальной сети Yatube. Позволяет создавать посты, комментировать их и подписываться на других пользователей.

## Особенности

- Аутентификация по JWT-токенам
- Создание и управление постами
- Комментирование постов
- Подписка на других пользователей
- Поиск по подпискам
- Права доступа: чтение доступно всем, изменение - только авторам
